Exercise 1: Run the Test App Locally

  1. Clone / Download git repo https://github.com/LaurensLeeuwis/training-gatling-testapp
  2. Run eu.leeuwis.training.gatling.testapp.TestApp from IntelliJ
  3. Open http://localhost:8080/

Gatling Recorder

Exercise 2: Run Gatling Recorder

  1. Clone / Download git repo https://github.com/LaurensLeeuwis/training-gatling-gatling
  2. Run Gatling Recorder
    Run Recorder from IntelliJ
  3. Change Proxy settings in your favorite browser
  4. Record scenario:
    • try to login with wrong credentials
    • Click through Angular App
    • Click through Plain HTML pages

Running the scenario

Exercise 3: Run recorded scenario

  1. Run Engine
    Run Engine from IntelliJ
  2. Report
    Check out target/results/x/index.html
  3. Errors?
    304 expected, got 200?

Looking at recordings

Exercise 4: Make scenario more useful

  1. Make sure that scenario runs perfectly
  2. Bundle calls to AngularJS app
    • Rename http request to index.html 'Angular App'
    • Rename http requests to javascript 'Angular App - Javascript'
    • Rename http requests to html views 'Angular App - Views'
  3. Rename http requests to simple HTML pages 'Simple pages'
  4. Run the test again. What changed?

Exercise 5: More users
